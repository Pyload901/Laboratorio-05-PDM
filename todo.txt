1. ¿Cuál es la principal ventaja de usar RecyclerView?
La principal ventaja es que optimiza la aplicación haciendo que el uso de recursos para la creación de elementos sea mínimo

2. Menciona los componentes de RecyclerView y la función que tiene cada uno de ellos
List Item: este componente es la vista que se mostrará por cada elementos creado, se trata de una plantilla que se poblará con los datos que se desee enviar

ViewHolder: este se encarga de guardar una referencia de cada view creada, de esta forma se asegura de poder reutilizar los componentes cuando sea necesario

Adapter: este componente sirve como un adaptador para que los datos se coloquen correctamente en la plantilla, es el que le dice al modelo de datos como debe mostrarse
